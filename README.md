# Pastelería – Backend (Spring Boot)

Backend del proyecto **Pastelería** desarrollado para la asignatura **Desarrollo Fullstack I (DSY1104)**.

## Tecnologías

- Java 17
- Spring Boot
- Spring Web
- Spring Data JPA
- Base de datos H2 en memoria

## Endpoints principales

- `GET /api/productos` – Lista todos los productos de la pastelería.
- `GET /api/productos/{id}` – Obtiene un producto por id.
- `POST /api/productos` – Crea un nuevo producto.
- `PUT /api/productos/{id}` – Actualiza un producto existente.
- `DELETE /api/productos/{id}` – Elimina un producto.
